# leaguesser
Leaguesser
O LeagueGuesser Pro é uma aplicação interativa voltada para testar o conhecimento do usuário sobre os campeões de League of Legends. O projeto conta com três modalidades de jogo distintas e uma interface otimizada para alto desempenho e fidelidade visual ao tema Hextech.

Funcionalidades Principais
Modo Clássico: Identificação do campeão através de comparação de atributos como alcance, gênero, posição, região e ano de lançamento. Inclui indicadores direcionais para o ano de estreia.

Modo Splash Art: Exibição de artes de skins aleatórias com aplicação de zoom dinâmico. O sistema de dicas permite a redução progressiva do zoom para facilitar a identificação.

Modo Habilidades: Teste de conhecimento mecânico onde o jogador deve identificar a tecla correspondente (Q, W, E ou R) de uma habilidade sorteada.

Sistema de Sequência: Contador integrado que monitora o número de acertos consecutivos em cada sessão.

Busca Dinâmica: Sistema de entrada de texto com sugestões em tempo real, exibindo o nome e o ícone do campeão para evitar erros de digitação.

Gerenciamento de Fluxo: Reinício automático do mini-game após a confirmação da vitória, sem necessidade de recarregamento da página ou retorno ao menu inicial.

Interface e Design
Layout Responsivo: Adaptável para diferentes resoluções de tela e dispositivos.

Identidade Visual: Paleta de cores baseada nas diretrizes de design da Riot Games, utilizando tons de dourado, azul hextech e carvão.

Feedback de Interface: Notificações visuais imediatas para acertos e erros, com exibição detalhada do campeão correto após cada rodada.

Especificações Técnicas
Linguagens: HTML5, CSS3 e JavaScript (ES6+).

Consumo de Dados: Integração com a CDN do Data Dragon para carregamento de ativos oficiais e atualizados.

Otimização: Código refatorado para garantir baixo tempo de resposta e manipulação eficiente do DOM.

Organização de Arquivos
index.html: Estrutura lógica e marcação da aplicação.

style.css: Definições de estilo, variáveis de cor e animações.

script.js: Lógica de processamento, controle de modos e validação de regras.

campeoes.js: Banco de dados local com as informações técnicas dos personagens.
